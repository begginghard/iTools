当前位置>Git命令大全>Git配置>config

1. 命令

config -- 配置 Git 相关属性，例如：用户信息，配色，编辑器等信息。

2. 使用样例

  • 配置全局用户名和邮箱

[123itools@shengqian ~]$ git config --global user.name begginghard
[123itools@shengqian ~]$ git config --global user.email begginghard@126.com
[123itools@shengqian ~]$ git config --list
user.name=begginghard
user.email=begginghard@126.com

  • 配置 git 输出结果加上颜色

[123itools@shengqian gitdemo]$ git config --global color.ui true
[123itools@shengqian gitdemo]$ git config --list
user.name=begginghard
user.email=begginghard@126.com
color.ui=true
core.repositoryformatversion=0
core.filemode=true
core.bare=false
core.logallrefupdates=true

  • 配置 git 常用操作的别名

[123itools@shengqian gitdemo]$ git config --global alias.ci commit
[123itools@shengqian gitdemo]$ git config --global alias.st status
[123itools@shengqian gitdemo]$ git config --global alias.co checkout
[123itools@shengqian gitdemo]$ git config --global alias.br branch
[123itools@shengqian gitdemo]$ git config --list
user.name=begginghard
user.email=begginghard@126.com
color.ui=true
alias.ci=commit
alias.st=status
alias.co=checkout
alias.br=branch
core.repositoryformatversion=0
core.filemode=true
core.bare=false
core.logallrefupdates=true

3. 使用方法

git config [选项]

git config 命令会将配置项写到配置文件中。系统级别的配置项写到 /etc/gitconfig，用户级别的配置项写到 ~/.gitconfig，项目级别的配置项写到 project/.gitconfig

4. 参数说明

  • --global 

设置用户级别的配置

  • --system

设置系统级别的配置

  • --local

设置库级别的配置

  • --list

显示当前配置了哪些属性
