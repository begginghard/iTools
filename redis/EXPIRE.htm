当前位置>Redis命令大全>Key(键)>EXPIRE

1. 命令 

        EXPIRE

            -- 为给定 key 设置生存时间，当 key 过期时(生存时间为 0 )，它会被自动删除

            -- 更新生存时间,对一个已经带有生存时间的 key 执行 命令，EXPIRE新指定的生存时间会取代旧的生存时间


2.使用说明

         EXPIRE key seconds

3.时间复杂度：O(1)

4.返回值： 设置成功返回 1 。当 key 不存在或者不能为 key 设置生存时间时，返回0

 3. 使用样例


redis> SET cache_page "www.google.com"

OK


redis> EXPIRE cache_page 30  # 设置过期时间为 30 秒

(integer) 1


redis> TTL cache_page    # 查看剩余生存时间

(integer) 23


redis> EXPIRE cache_page 30000   # 更新过期时间

(integer) 1


redis> TTL cache_page

(integer) 29996
