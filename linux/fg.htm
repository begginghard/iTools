1. 命令

fg -- 将后台中的程序调到前台继续执行

2. 使用样例

  • 默认将后台中的带 "+" 的进行调至到前台继续执行

[root@shengqian tmp]# jobs
[1]-  已停止               vim a
[2]+  已停止               vim b
[root@shengqian tmp]# fg
vim b

  • 选择后台中的某个进程调至到前台运行

[root@shengqian tmp]# jobs
[1]-  已停止               vim a
[2]+  已停止               vim b
[root@shengqian tmp]# fg %1
vim a

[root@shengqian tmp]# jobs
[1]+  已停止               vim a
[2]-  已停止               vim b
[root@shengqian tmp]# fg %2
vim b

3. 使用方法

fg [%number]

4. 参数说明

  • %number

当后台有多个进程时，通过指定 number 来唤醒某一个进程
