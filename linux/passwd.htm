当前位置>Linux命令大全>用户管理>passwd

1. 命令

passwd -- 设置某个用户的登录密码

2. 使用样例

  • 对 123itools 用户名设置登录密码。【root 用户功能】

[root@shengqian ~]# passwd 123itools
更改用户 123itools 的密码 。
新的 密码：
重新输入新的 密码：
passwd：所有的身份验证令牌已经成功更新。

  • 修改当前用户的登录密码

[123itools@shengqian ~]$ passwd
更改用户 123itools 的密码 。
为 123itools 更改 STRESS 密码。
（当前）UNIX 密码：
新的 密码：
重新输入新的 密码：
passwd：所有的身份验证令牌已经成功更新。

  • 查看123itools用户密码的状态

[root@shengqian ~]# passwd -S 123itools
[root@shengqian ~]# passwd -S 123itools
123itools PS 2016-11-02 0 99999 7 -1 (密码已设置，使用 SHA512 算法。)

每列内容分别为：用户，密码状态，设置密码时间 2016-11-02，0天内不可以修改密码，更改密码天数 99999，
密码失效前 7 天警告，永久不失效.

  • 设置 123itools 的密码，并设置 1 天内不可以修改密码。【root 功能】

[root@shengqian ~]# passwd 123itools -n 1
调整用户密码老化数据123itools。
passwd: 操作成功

[root@shengqian ~]# passwd -S 123itools
123itools PS 2016-11-02 1 99999 7 -1 (密码已设置，使用 SHA512 算法。)

[123itools@shengqian ~]$ passwd
更改用户 123itools 的密码 。
为 123itools 更改 STRESS 密码。
（当前）UNIX 密码：
您必须等待更长时间以更改密码
passwd: 鉴定令牌操作错误

  • 设置 123itools 的密码，并设置密码失效日期为 20161103，并且提前 1 天警告。

[root@shengqian ~]# passwd -w 1 -i 20161103 123itools
调整用户密码老化数据123itools。
passwd: 操作成功
[root@shengqian ~]# passwd -S 123itools
123itools PS 2016-11-02 1 99999 1 20161103 (密码已设置，使用 SHA512 算法。)

  • 锁定 123itools 用户密码，使其无法登陆。

[root@shengqian ~]# passwd -l 123itools
锁定用户 123itools 的密码 。
passwd: 操作成功

  • 解锁 123itools 用户密码，使其可以登录。

[root@shengqian ~]# passwd -u 123itools
解锁用户 123itools 的密码。
passwd: 操作成功

3. 使用说明

passwd [-l] [-u] [-S] [-n 天数] [-x 天数] [-w 天数] [-i 日期] 账号

4. 参数说明

  • -l

锁定某个用户密码，使其不可以登录

  • -u

解锁某个用户密码，使其可以登录

  • -S

查看某个用户的密码状态

  • -n 天数

表示多久不可以修改密码天数。

  • -x 天数

多久内必须要改动密码。

  • -w 天数

密码过期前的警告天数

  • -i 日期

设置密码过期日期

  • /etc/shadow

shadow 文件用来保存用户的密码信息，下面是 shadow 文件的样例 以及 每列含义：

123itools:$6$fm7zxyP1$YJSHjr3NimZe2USnWXP7sDgsBSw0hgXRqfrLIim8oB5B7lfelv1qxqu9xnLJBuvD/kfrYJ9LapKa0CqH1cdlI/:17107:1:99999:1:20161103::

账号名称；

密码：这个字段内的数据才是真正的密码，而且是加密后的密码。

最近更改密码的日期：该日期是以 1970年1月1日起按天来累计的天数。

密码不可被更改的天数。

密码必须更改的天数：强制要求用户修改密码，否则这个密码将会过期。

密码需要更改期限前的警告天数：当前密码有效期限快到的时候，系统会一句这个字段设置发出“警告”，提醒这个账号再过 n 天密码就要过期了，请重新设置你的密码。

密码过期后的账号宽限时间：即密码失效期。

账号失效日期：这个账号在此字段规定的日期之后，将无法再使用。
