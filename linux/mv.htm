1. 命令

mv -- 移动或重命名文件

2. 使用样例

  • 重命名文件，将 test1 重命名为 test2

mv test1 test2

  • 移动文件，将 test1 移动到 test 目录下

mv test1 test/

  • 移动文件，替换已存在文件；将文件 test1 移动到 test 目录下，如果目录中存在 test1 文件，强制替换

mv -f test1 test/

  • 移动文件，不替换已存在文件；将文件 test1 移动到 test 目录下，如果目录中存在 test1 文件，不替换

mv -n test1 test/

3. 使用方法

mv [属性] [源文件] [目的文件]     将源文件重命名为目的文件

mv [属性] [源文件] [目录]            将源文件移动到目录下

4. 参数说明

  • -f

强迫替换目的目录中已存在的文件

  • -n 

不替换目的目录中已存在的文件
