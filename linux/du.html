1. 命令

du -- 显示某个文件的磁盘使用空间，如果是目录的话，递归显示。

2. 使用样例

  • 显示某个文件的占用磁盘空间大小。例如 /path/file

du /path/file

  • 显示某个目录下所有子目录的占用磁盘空间大小，递归子目录。例如：/path/

du /path/

  • 显示某个目录所有目录和文件占用磁盘空间大小。

du -a /path/

  • 显示某个目录下所有文件占用磁盘空间总和。

du -s /path/

  • 显示某个目录下一级子目录占用磁盘空间情况。

du -d 1 /path/

  • 显示某个目录下一级子目录和文件占用磁盘空间情况。

du -ad 1 /path/

  • 将磁盘空间大小显示为方便阅读的。

du -h /path/

  • 显示磁盘空间大于 1G 的文件

du -t +1G /path/

  • 显示磁盘空间小于 1G 的文件

du -t -1G /path/

3. 使用方法

du [属性]... [文件]...

4. 参数说明

  • -a, --all 

输出所有文件和目录

  • -s, --summarize

显示目录或文件的总大小

  • -d,--max-depth=N

递归到第 N 级子目录

  • -h,--human-readable

将磁盘空间大小显示为方便阅读的

  • -t, --threshold=SIZE

按文件大小过滤，如果 SIZE 为正，则显示大于 SIZE 的文件；如果 SIZE 为负，则显示小于 SIZE 的文件。
