1. 命令

top -- 展现 Linux 进程状态

2. 使用样例

  • 展现当前系统资源使用情况 和 各进展使用系统资源的状态

top

  • 设置刷新频率为 1s

top -d 1 

top 之后，直接按 d 去设置刷新间隔时间

  • 按照 MEM 使用大小排序

top 之后，按大写的 M 

  • 按照 CPU 使用大小排序

top 之后，按大写的 P

  • 仅观察 pid 为 1234 的进程

top -p 1234

  • 观察 pid 为 1234, 2234 的进程执行状态

top -p 1234,2234

  • 只展现 user 为 root 的进程

top -u root

top 之后，按 u 去设置用户信息

3. 使用方法

top [属性]....

4. 参数说明

  • -d

刷新间隔，单位为 s

  • -p 

指定 pid 过滤进程

  • -u

指定用户名过滤进程
