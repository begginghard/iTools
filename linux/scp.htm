1. 命令

scp -- 远程拷贝文件，即将某一台机器的文件拷贝到另外一台机器上

2. 使用样例

  • 将本机 file 文件拷贝到 43.241.208.129 服务器 zgh 用户的 /tmp 目录下

scp file zgh@43.241.208.129:/tmp/

  • 将本机 /path 目录下的所有文件拷贝到 43.241.208.129 服务器 zgh 用户的 /tmp 目录下

scp -r /path zgh@43.241.208.129:/tmp/

  • 将 43.241.208.129 服务器 zgh 用户的 /tmp/file 文件拷贝到本机当前目录

scp zgh@43.241.208.129:/tmp/file .

  • 将 43.241.208.129 服务器 zgh 用户的 /path 目录完整拷贝到本机当前目录

scp -r zgh@43.241.208.129:/path .

3. 使用方法

scp [属性]... 源文件 目的文件

4. 参数说明

  • -r

递归子目录
