1. 命令

cp -- 复制文件到当前目录，复制文件到某个目录下

2. 使用样例

  • 复制文件到当前目录下，例如将 test1 复制一份命名为 test2

cp test1 test2

  • 复制文件到某个目录下，例如将 test1 复制到 test 目录下

cp test1 test/

  • 复制文件到某个目录下，并覆盖已存在的文件

cp -f test1 test/

  • 递归复制子目录，将 test1 目录下的所有子目录均复制到 test 下

cp -r test1/ test/

  • 为某个文件创建符号链接，为 test1 创建一个符号链接 test2

cp -s test1 test2

3. 使用方法

cp [属性]... [源文件] [目录]/[文件]

4. 参数说明

  • -f

强制覆盖已存的文件

  • -r 

递归复制子目录下的文件

  • -s

创建符号链接
