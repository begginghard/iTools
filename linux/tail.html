1. 命令

tail -- 显示文件的最后一部分内容

2. 使用样例

  • 持续查看某个文件的追加写入数据。

tail -f file

  • 显示某个文件的最后 1000 字节的数据

tail -c 1000 file

  • 显示某个文件的最后 1000 行数据

tail -n 1000 file

3. 使用方法

tail [属性]... [文件]...

4. 参数说明

  • -f --follow[=name|descriptor]

持续输出追加写入该文件的数据

  • -F 

与 --follow=name 相同，如果有文件删除操作不中断

  • -c --bytes=K

输出最后 K 字节的数据

  • -n --lines=K

输出最后 K 行的数据
