1. 命令

yum -- 在 Redhat, CentOs 中用来管理 rpm 包的查找，安装，更新，卸载，依赖等问题。

2. 使用样例

  • 安装 mysql

安装 mysql server

#yum install mysql-community-server.x86_64


  • 安装过程中默认都选择 yes

#yum -y install mysql-community-server.x86_64


  • 卸载 mysql server

#yum erase mysql-community-server.x86_64


  • 升级 mysql server

#yum update mysql-community-server.x86_64


3. 使用方法

yum [options] command package

command

  • help

显示用法提示

  • search

在软件包详细信息中搜索指定字符串

  • install

向系统中安装一个或多个软件包

  • update

更新系统中的一个或多个软件包

  • repolist

显示已配置的源

  • reinstall

覆盖安装软件包

  • deplist

列出软件包的依赖关系

  • check-update

检查是否有可用的软件包更新

  • info

显示关于软件包或组的详细信息

options

  • -h, --help

显示此帮助消息并退出

  • --version

显示 Yum 版本然后退出

  • -y, --assumeyes

回答全部问题为是

  • -v, --verbose

显示详尽的操作过程

  • -q, --quiet

静默执行

  • -c [config file], --config=[config file]
