1. 命令

useradd -- 新增一个用户

2. 使用样例

  • 查看当前系统所有用户

[root@shengqian ~]# cat /etc/passwd
root:x:0:0:root:/root:/bin/bash
bin:x:1:1:bin:/bin:/sbin/nologin
daemon:x:2:2:daemon:/sbin:/sbin/nologin
adm:x:3:4:adm:/var/adm:/sbin/nologin
lp:x:4:7:lp:/var/spool/lpd:/sbin/nologin
sync:x:5:0:sync:/sbin:/bin/sync
shutdown:x:6:0:shutdown:/sbin:/sbin/shutdown
halt:x:7:0:halt:/sbin:/sbin/halt
mail:x:8:12:mail:/var/spool/mail:/sbin/nologin
operator:x:11:0:operator:/root:/sbin/nologin

  • 新增一个用户，指定 UID 和 GID

[root@shengqian ~]# useradd -u 600 -g users zgh
[root@shengqian ~]# cat /etc/passwd | grep "zgh"
zgh:x:600:100::/home/zgh:/bin/bash

3. 使用方法

useradd [-u UID] [-g 初始用户组] [-G次要用户组] [-mM] [-c 说明栏] [-d 主文件夹的绝对路径] [-s shell] 用户账号名

4. 参数说明

  • /etc/passwd

[root@shengqian ~]# cat /etc/passwd | grep "123itools"
123itools:x:1002:1003::/home/123itools:/bin/bash

账号名称:  对应 UID。例如 root 的 UID 对应的就是 0。

密码:  早期 UNIX 系统的密码就是放到这个字段上，但是因为这个文件的特性是所有程序都能够读取，故将密码修改到 /etc/shadow，在这里你只能看到一个 X

用户标识符：0 系统管理员；1~499 系统账号；500~65535 可登陆账号

GID: 组 ID

用户信息说明列： 解释账号的意义

Shell: 登录 shell


  • -u

UID，指定固定的 UID

  • -g 

group，指定该用户所属的 group，或者 gid

  • -c 

指定 /etc/passwd 第 5 列，解释账号的意义

  • -d

指定主目录文件夹，即指定 /etc/passwd 第6列，登录该用户之后默认的文件夹。

  • -s shell

后面接一个 shell，不指定默认 /bin/bash。
