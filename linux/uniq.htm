1. 命令

uniq -- 统计或删除相邻相同内容的行

2. 使用样例

  • 将 file 文件内容相邻相同的行去掉，只保留一行

uniq file

cat file | uniq

  • 将 file 文件中内容相同的行去掉，只保留一行

cat file | sort | uniq

sort file | uniq

  • 将 file 文件中内容相同的行去掉，只保留一行，并统计行信息

cat file | sort | uniq -c

sort file | uniq -c

  • 仅显示 file 文件中内容相同的行

cat file | sort | uniq -d

  • 仅显示 file 文件中内容不同的行

cat file | sort | uniq -u

  • 显示文件 a 和 文件 b 的交集

cat a | sort | uniq > a_sort

cat b | sort | uniq > b_sort

cat a_sort b_sort | sort | uniq -d

  • 显示文件 a 和 文件 b 的并集

cat a | sort | uniq > a_sort

cat b | sort | uniq > b_sort

cat a_sort b_sort | sort | uniq

  • 显示文件 a 和 文件 b 的差集

cat a | sort | uniq > a_sort

cat b | sort | uniq > b_sort

cat a_sort b_sort b_sort | sort | uniq -u

3. 使用方法

uniq [属性].... [输入 [输出]]

4. 参数说明

  • -c, --count

将相同内容的行数信息打印出来

  • -d, --repeated

只打印相邻内容重复的行数

  • -u, --uniq

只打印唯一行内容的行数
