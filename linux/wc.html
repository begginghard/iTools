1. 命令

wc -- 输出文件的行数，单词个数，字节数

2. 使用样例

  • 显示某个文件的行数，单词数，字节数

wc file

  • 显示某个文件的行数

wc -l file

  • 显示某个文件的单词个数

wc -w file

  • 显示某个文件的字节个数

wc -c file

  • 显示文件中最长的一行的字节数

wc -L file

3. 使用方法

wc [属性]... [文件]...

4. 参数说明

  • -c, --bytes

显示字节数

  • -w, --words

显示单词数

  • -l, --lines

显示行数

  • -L, --max-line-length

显示最长行的字节数
