1. 命令

iostat -- 展现 CPU 使用状态 和 磁盘 IO 状态

2. 使用样例

  • 展现当前 CPU 使用状态 和 磁盘 IO 状态

iostat

  • 连续展现当前 CPU 使用状态 和 磁盘 IO 状态，间隔时间为 2s

iostat 2

  • 展现当前 CPU 使用状态 和 磁盘 IO 状态，间隔时间为 2s，总共展现次数为 10 次

iostat 2

  • 仅展现当前 CPU 使用状态

iostat -c

  • 仅展现当前 磁盘 IO 的使用状态

iostat -d

  • 使用方便人阅读的方式展现磁盘 IO 的使用状态

iostat -d -h

  • 使用 MB 单位展现磁盘 IO 的使用状态

iostat -d -m

  • 使用 KB 单位展现磁盘 IO 的使用状态

iostat -d -k

3. 使用方法

iostat [属性]... [间隔秒数] [展现次数]

4. 参数说明

  • -c

仅展现 cpu 状态

  • -d

仅展现 磁盘 io 状态

  • -h

以方便人阅读的方式展现

  • -k

以 kb 为单位展现磁盘 IO 的使用状态

  • -m

以 mb 为单位展现磁盘 IO 的使用状态
