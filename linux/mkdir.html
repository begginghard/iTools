1. 命令

mkdir--创建目录

2. 使用样例

  • 创建一个空目录为 test1

mkdir test1

  • 创建一个多级目录为 test1/test2/test3

mkdir -p test1/test2/test3

  • 一次创建多个多级目录, test1/test11, test1/test12

mkdir -p test1/{test11, test12}

  • 一次性创建多个多级目录，test/test1, test/test1/test11, test/test2, test/test2/test21, test/test2/testt22, test/test2/test22/test222

mkdir -p test/{test1/{test11, test12}, test2/{test21, test22/{test222}}}

  • 创建属性为 777 的目录，test

mkdir -m 777 test

3. 使用方法

mkdir [属性]... [目录名]...

4. 参数说明

  • -p

自动创建该目录路径中的父目录

  • -m --mode=MODE

创建属性为 MODE 的 目录
