1. 命令

head -- 显示文件的前部分内容

2. 使用样例

  • 显示某个文件的前部分内容，例如查看 file 前10行内容

head file

  • 显示某个文件的前1000字节内容

head -c 1000 file

  • 显示某个文件的前1000行内容

head -n 1000 file

3. 使用方法

head [属性]... [文件]...

4. 参数说明

  • -c --bytes=[-]K

显示每个文件的前 K 个字节的内容；如果加 - 的话，则输出除了最后 K 个字节的所有内容。

  • -n --lines=[-]K

显示每个文件的前 K 行的内容；如果加 - 的话，则输出除了最后 K 行的所有内容。
