1. 命令

chmod -- 每个文件有三个权限范围: user(u)，group(g)，other(o)，all(a)。文件权限分为三种: read(r)，write(w)，execute(x) 等权限。可以更改不同权限范围内的权限。

2. 使用样例

  • 为 file 的 user, group, other 全部增加 可执行(x) 权限

chmod a+x file

  • 为 file 的 user, group, other 全部移除 可执行权限

chmod a-x file

  • 为 file 的 user 赋予读写可执行权限，为 group 赋予读可执行权限，other 仅赋予读权限

chmod u=rwx,g=rx,o=r file

chmod 764 file

  • 设置 file 为所有人可读，可写，可执行

chmod a=rwx file

chmod 777 file

  • 将某个目录下所有文件都设置为 777 权限

chmod -R 777 /path

3. 使用方法

chmod [属性]... 权限[,权限]... [文件]...

4. 参数说明

  • u

user，指该文件的拥有者

  • g

group，指该文件所属组

  • o

other，指其他人

  • a

所有人

  • r

读权限 = 4

  • w

写权限 = 2

  • x

可执行权限 = 1

  • -R, --recursive

递归操作
