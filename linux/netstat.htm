1. 命令

netstat -- 打印网络连接

2. 使用样例

  • 打印当前所有处于 LISTEN 状态的连接

netstat -l

  • 打印当前所有处于 LISTEN 和 ESTABLISHED 状态的连接

netstat -a

  • 输出该连接所属的 pid

netstat -ap

  • 仅输出 TCP 协议的连接

netstat -atp

  • 仅输出 UDP 协议的连接

netstat -utp

3. 使用方法

netstat [属性]...

4. 参数说明

  • -l, --listening

仅仅输出处于 LISTEN 状态的连接

  • -a, --all

显示处于 LISTEN 和 ESTABLISHED 状态的连接

  • -p, --pid

输出连接所属的进程 id

  • -t, --tcp

输出 tcp 协议的连接

  • -u, --udp

输出 udp 协议的连接
