1. 命令

ping -- 验证与某台服务器连通情况

2. 使用样例

  • 验证本机与 43.241.208.129 网络的联通状态

ping 43.241.208.129

  • 验证本机与 www.123itools.com 的网络联通状态

ping www.123itools.com

  • 验证本机与 43.241.208.129 网络联通状态，仅发送 100 个包，且每次间隔 5s

ping -c 100 -i 5 43.241.208.129

3. 使用方法

ping [属性]... [目的地]

4. 参数说明

  • -c count

发送 count 个包之后，停止发送

  • -i interval

每隔 interval 秒之后再次发送一个包
