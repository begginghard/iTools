1. 命令

chown -- 改变文件的 所属用户 和 所属组

2. 使用样例

  • 修改 file 的 owner 为 123itools

chown 123itools file

  • 修改 file1, file2 的 所属用户 为 123itools

chown 123itools file1 file2

  • 修改 file 的 所属用户 为 123itools，所属组为 123itools

chown 123itools:123itools file

  • 递归修改某个目录下所有文件的所属用户为 123itools。

chown -R 123itools file

3. 使用方法

chown [属性]... [所属用户][:[所属组]] [文件]...

4. 参数说明

  • -R, --recursive

操作文件，且递归操作子目录
