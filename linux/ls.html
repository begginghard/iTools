1. 命令

 ls -- 显示文件信息

2. 使用样例

  • 显示当前目录所有文件信息(包括隐藏文件)

ls -al

  • 显示当前目录所有文件信息(包括可读的文件大小)

ls -alh

  • 显示当前目录所有文件信息，并按文件从大到小排序

ls -alS

  • 显示当前目录下目录信息

ls -d

  • 递归显示该目录下的所有文件

ls -alR

  • 显示当前目录所有文件信息，并按文件从小到大排序

ls -alSr

3. 使用方法

ls [选项]... [文件]...

4. 参数说明

  • --help

显示命令使用方法

  • -a

显示所有文件，包括隐藏文件

  • --block-size=Size

按照 Size 展示文件大小，Size 可以使用 G, M, K

  • -d

仅显示目录

  • -l

显示文件的详细信息，包括文件权限，所属用户，所属用户组，大小，创建时间，文件名

  • -r --reverse

反序显示文件

  • -R --recursive

递归显示子目录

  • -S

按照文件大小排序
